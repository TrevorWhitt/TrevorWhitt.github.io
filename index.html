Biography

Hi! Glad to see you here. I'm a GIMM (Games, Interactive Media, and Mobile technology) student at Boise State University expecting to graduate in the Spring of 2028, and a budding game developer and website developer. This is my fourth semester of GIMM classes, and I've had loads of opportunities to develop games and build websites. I have made one website in full for a project in a previous semester, and this website is my official portfolio website, wherein I will be showing off all of my portfolio artifacts, projects, and WIPs. I have also made a number of games/stubs games, including a platformer and an AR game. I am eagerly awaiting the upcoming solo game project, which I will be designing and creating in Unreal Engine 5.
Prior to my time in the GIMM program, I have had zero experience in game development, so this has been a fabulous opportunity to build my skills.

School Projects

These are the projects I have worked on or been involved in as assignments and projects in GIMM.

The Emptiness of an Atom - Interactive Comic

This comic was about the parts of atoms that are empty. About 99% of an atom's volume is empty space, even though it really isn't. It had a lot to do with quantum mechanics and it was a group project. It taught me that you won't always get along with the people you work with, and yet you can still finish a project. I composed all of the game's soundtrack, did all of the voice acting, and comtributed the first 4[????] background panels. What was most enjoyable about this project was getting involved in aspects of game development that I hadn't delved into before (not to any great length, anyways); aspects like scoring, sound mixing, and creating art assets.

AR Blackjack - AR Game

This game was designed with augmented reality in mind. It was another group project, and we mixed individual game projects we had been working on with blackjack to create an AR game which used the cards you pulled for blackjack to change the way you played the game. I learned a lot about deadlines and team work with this project. What I really enjoyed was sitting with the group and struggling our way through the process of combining code developed by multiple people and somehow coming out the other side with a (marginally) working project.

Individual Game Project titled "Descent into Darkness" - Proof of Concept Game Demo

This game was my first ever game project. I was given a stub game and using the provided foundation, I had to make my own game out of it using "physics changes," which were things like adding a double jump or a slow fall. It was a life-changing experience giving me the confidence to know with enough willpower and dedication I could actually put code and art assets together to make an interactive game.
